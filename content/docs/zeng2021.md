## Summary
MusicBERT is inspired by BERT but uses a masking strategy that is music-unique since certain masking techniques are too easy for music (guessing adjacent token values of time signature or tempo). The new masking strategy is bar-level and masks out same types of elements, where 80% are MASK tokens, 10% random, and 10% not changed.

OctupleMIDI is an input encoding with the 8 elements of time signature (fraction), tempo (24-200), bar and position (0-255), instrument (MIDI), pitch (128), duration (multi-scale for greater resolution at small durations), and velocity (32 values).

Compiles a new Million-MIDI Dataset that is deduplicated according to everything beside instrument and pitch.

## Key Contributions
- Develops a new input format OctupleMIDI
- Develops a unique bar-level masking strategy
- Develops a new dataset Million-MIDI Dataset
